# 🎂 Birthday Surprise Project – Instructions

## Project Name

**Birthday Surprise Website**

## Goal

Create a small romantic website that acts as a digital birthday gift.
The website will include animations, photos, a message, a quiz, and a final surprise message for your girlfriend.

---

# 🧩 Project Concept

The website works like a small interactive story.

1. She opens the link.
2. A beautiful birthday greeting appears.
3. She clicks a button to continue.
4. Your memories and photos appear.
5. She plays a small quiz about your relationship.
6. A final emotional message appears.

---

# 🧠 Quiz Section – “How Well Do You Know Me?”

Create a small interactive quiz to make the website fun and personal.

### Concept

The quiz should contain questions about:

• your relationship
• your favorite things
• memories you shared

Example idea:

```
Question 1: [You will add later]

A)
B)
C)
D)
```

### Instructions

1. Create a quiz section in the website.
2. Show one question at a time.
3. Allow her to select an answer.
4. Move to the next question after answering.
5. Show the final score at the end.

### Result Messages

If score is high:

```
Wow ❤️ You know me perfectly!
```

If score is medium:

```
Pretty good 😄 but you still need to learn more about me.
```

If score is low:

```
Haha 😆 we need more memories together!
```

### Important

You will **add the real quiz questions later**.
